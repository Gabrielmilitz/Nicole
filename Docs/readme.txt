Nicole is a bot that:

Understands user questions using embedding models (sentence-transformers).

Learns new answers over time (self-learning).

Searches for information in PDF documents uploaded to the system.

Generates realistic images based on prompts using the Hugging Face API.

Responds naturally, with varied phrases and emojis.

Features a futuristic neon visual style (inspired by modern ChatGPT interfaces).

Interacts through the web browser, without needing page reloads.

Stores the user's name and personalizes the entire experience.
